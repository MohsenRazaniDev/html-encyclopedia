What is HTML?

HTML stands for HyperText Markup Language and is the standard language for creating web pages.

Semantic HTML

Semantic elements clearly describe their meaning in a human- and machine-readable way.

  • <header>, <footer>, <article>, <section>
  • Help improve accessibility and SEO

Basic HTML Document Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <!-- Content goes here -->
  </body>
</html>
