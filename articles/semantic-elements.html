What Are Semantic Elements?

Semantic elements clearly define the meaning and structure of web content, making it easier for browsers, developers, and assistive technologies to understand.

Common Semantic Tags

  • <header> - Defines a header for a document or section
  • <nav> - Defines navigation links
  • <article> - Defines an independent, self-contained piece of content
  • <section> - Defines a thematic grouping of content
  • <aside> - Defines content aside from the main content
  • <footer> - Defines a footer for a document or section

Benefits of Using Semantic Elements

Semantic HTML enhances:

  • Accessibility: Screen readers can navigate content more effectively
  • SEO: Search engines better understand your page structure
  • Maintainability: Clear structure helps developers manage code easily

Example Usage

<header>
  <h1>Page Title</h1>
</header>
<nav>
  <ul>
    <li><a href="#">Home</a></li>
  </ul>
</nav>
<article>
  <h2>Article Heading</h2>
  <p>Article content...</p>
</article>
<footer>
  <p>Footer information</p>
</footer>
